User-Interface

Diese Seite soll die unterschiedlichen Ansichten des NagVis-User-Interfaces beschreiben. Die Pfade sind relativ zum NagVis-root-Verzeichnis.

Mit NagVis 1.5 wurde das URL-Schema verändert. Falls Ihre alten URLs weiterhin in NagVis 1.6 funktionieren sollen, dann können Sie die mod_rewrite-Konfiguration in der mitgelieferten Beispiel-Webserver-Konfiguration nutzen.

URLBeschreibung
NagVis Frontend
index.phpDies ist eine Weiterleitung zu nagvis/index.php (NagVis-Frontend)
nagvis/frontend/nagvis-js/index.phpDies ist der Map-Index, die Hauptseite von NagVis. Auf der Hauptseite sind alle Maps aufgeführt, für die der Benutzer berechtigt ist. Sie können eine Map ausschließen, indem Sie die show_in_lists-Option auf 0 setzen.
nagvis/frontend/nagvis-js/index.php?mod=Map&show=<name>Dies ist die klassische Map-Ansicht. In diesem Modus wird die angegebene Map angezeigt und mit Hilfe von Ajax-Aufrufen an den NagVis-Core aktualisiert.

Die folgenden Parameter sind verfügbar:

search=<regex>Suchen nach einem Map-Objekt während des Ladevorgangs. Gefundene Objekte werden hervorgehoben.
rotation=<name>Rotieren der Objekte im angegebenen Rotations-Pool, wenn die Map Bestandteil des Pools ist.
enableHeader=1/0Aktivieren/Deaktivieren des Header-Menüs, unabhängig von der Konfigurationseinstellung
enableHover=1/0Aktivieren/Deaktivieren des Hover-Menüs, unabhängig von der Konfigurationseinstellung
enableContext=1/0Aktivieren/Deaktivieren des Context-Menüs, unabhängig von der Konfigurationseinstellung
nagvis/frontend/nagvis-js/index.php?mod=Automap&show=<name>Dies ist die Automap-Ansicht. In dieser Ansicht wird die angegebene NagVis-Map angezeigt. Der Inhalt wird automatisch mit Hilfe von Ajax-Aufrufen an den NagVis-Core aktualisiert.

Die verfügbaren Parameter finden Sie auf der Automap-Seite.
nagvis/frontend/nagvis-js/index.php?infoDies ist eine Seite mit Support-Informationen, die Sie mitliefern können, wenn Sie Probleme mit NagVis haben.
Rotationen
nagvis/frontend/nagvis-js/index.php?rotation=<name>Dies ist der Map-Rotationsmodus. In diesem Modus wird die erste angegebene Map angezeigt (Rotation muss in der Hauptkonfigurationsdatei definiert werden). Nach einem definierten Intervall wird die nächste angegebene Map geladen usf.
nagvis/frontend/nagvis-js/index.php?rotation=<name>&rotate=0Dies ist der Rotationsmodus. Die Rotation wird durch den angegebenen Wert gestoppt. In diesem Modus werden die Maps automatisch aktualisiert. Es kann sinnvoll sein, eine Rotation zu stoppen, wenn Sie etwas Interessantes auf der Map gefunden haben. Die Rotation wird aktiviert, wenn Sie den Wert auf 1 setzen oder den Rotationswert löschen.
WUI (NagVis-Konfiguration)
config.phpDies ist eine Weiterleitung zu wui/index.php (Konfigurations-Interface)
wui/index.phpDies ist die Willkommen-Seite des NagVis-Konfigurations-Interfaces (WUI). Rechtsklicken Sie auf dem Hintergrund-Bild, um das WUI-Menü zu öffnen.
